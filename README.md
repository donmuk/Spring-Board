# 게시판 만들기 Project

## 프레임워크
- Spring, MyBatis

## 언어
- Java, JavaScript, SQL

## 개발툴
- Eclipse, SQL Developer

## 사용 UI Lib
- BootStarp

## 구현기능 목록

### 게시글
- 목록
- 작성
- 수정
- 삭제
- 조회
- 삭제 벨리데이션
- 페이징 처리
- 검색
- 페이징 검색 유지
- 조회 수(트랜잭션 처리)

### 게시글 댓글
- 댓글 목록
- 댓글 작성
- 댓글 수정 
- 댓글 삭제 

### 회원가입
- 로그인
- 회원가입 아이디 중복 체크
- 회원가입 비밀번호 암호화
- 회원정보 수정
- 회원탈퇴 
- 회원탈퇴 비밀번호 체크
